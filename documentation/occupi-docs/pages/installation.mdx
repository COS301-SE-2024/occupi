## 1. Mobile App Installation

The Occupi mobile app is a companion app that offers office occupancy prediction. Installation methods differ for Android and iOS devices.

### Android Installation

1. On your Android device, open a web browser and navigate to the Occupi download page (e.g., https://occupi.tech/download).
2. Click on the Android APK download link.
3. If prompted, allow your browser to download APK files.
4. Once the APK is downloaded, tap on the notification or go to your Downloads folder.
5. Tap on the APK file to begin installation.
6. If asked, allow installation from unknown sources in your device settings.
7. Follow the on-screen prompts to complete the installation.
8. Once installed, tap "Open" to launch the app.
9. Follow the in-app instructions to set up your account and configure the app.

### iOS Installation

1. On your iOS device, download the Expo Go app from the App Store if you haven't already.
2. Open your device's camera app.
3. Navigate to the Occupi QR code page (e.g., https://occupi.tech/ios-qr).
4. Scan the QR code using your camera.
5. When prompted, tap to open the link in Expo Go.
6. The Occupi app will load within Expo Go.
7. Follow the on-screen instructions to set up your account and configure the app.

## 2. Web Interface Setup

The Occupi web interface is an admin portal used to view more analytical data for all workers. To set up and access the web interface:

1. Open a web browser on your computer.
2. Navigate to https://dev.occupi.tech
3. Click on the "Sign Up" or "Create Account" button.
4. Fill in the required information to create an admin account.
5. Verify your email address if required.
6. Log in to the admin portal using your newly created credentials.
7. Follow the on-screen instructions to set up your organization and configure the dashboard.

## 3. System Requirements

### Mobile App
- Android: Version 8.0 (Oreo) or later
- iOS: Version 13.0 or later, Expo Go app installed
- Stable internet connection (Wi-Fi or cellular data)
- Location services enabled

### Web Interface
- Modern web browser (Chrome, Firefox, Safari, or Edge; latest two versions)
- JavaScript enabled
- Minimum screen resolution: 1280x720
- Stable internet connection

## 4. Troubleshooting

If you encounter any issues during installation or setup, try the following steps:

1. Ensure your device meets the minimum system requirements.
2. Check your internet connection.
3. Clear your browser cache and cookies (for web interface issues).
4. For Android: Try downloading the APK file again and reinstalling.
5. For iOS: Ensure you have the latest version of Expo Go installed.
6. Try scanning the QR code again or manually entering the provided URL in Expo Go.

If problems persist, please contact our support team at support@occupi.tech or visit our help center at https://help.occupi.tech for further assistance.
